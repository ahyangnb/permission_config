# Permission config.
to implement permission like:
```yaml
permission_config:
  android:
    store: true
    location: true
    camera: true
    internet: true

  ios:
    store: true
    storeTip: "Hi i wanna store some data"
    location: true
    locationTip: "Hi i wanna store some data"
    camera: true
    cameraTip: "Hi i wanna store some data"
    internet: true
    internetTip: "Hi i wanna store some data"

```
